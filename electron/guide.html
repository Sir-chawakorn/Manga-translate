ภาพรวมหน้าจอ

หน้าจอหลักประกอบด้วย 3 ส่วน: แถบเครื่องมือด้านบน, พื้นที่ทำงานตรงกลาง, และแผงแก้ไขด้านขวา

2

เปิดไฟล์

กดปุ่ม เปิดไฟล์ หรือ Ctrl+O เพื่อเลือกไฟล์:

  • ภาพ — .png, .jpg, .jpeg, .webp, .gif, .bmp
  • PDF — เปิดแล้วเลือกหน้าที่ต้องการแปลจากแผงด้านซ้าย
  • โปรเจค — .mtsproj เพื่อเปิดงานที่บันทึกไว้

Edit Mode (แก้ไขด้วย AI)

ในหน้า Preview กดปุ่ม Edit Mode เพื่อระบายจุดที่ต้องการแก้ไขเพิ่มเติม:

แถบเครื่องมือ Edit Mode
Brush: 20px Undo Clear
ส่งให้ Gemini แก้ไข
  1. ระบายสีแดง ทับบริเวณที่ต้องการแก้ไข (opacity 50%)
  2. พิมพ์ prompt บอก Gemini ว่าต้องการทำอะไร
  3. กด ส่งให้ Gemini แก้ไข
  4. รอ Gemini ประมวลผล → ดูผลก่อน-หลังด้วย slider
  5. ถ้ายังไม่พอใจ ระบายเพิ่มแล้วส่งอีกรอบได้
Gemini เห็นสิ่งที่คุณระบาย
ภาพที่ส่งไปจะมีรอยระบายสีแดงอยู่ Gemini จะเข้าใจว่าต้องแก้ไขตรงไหนจาก prompt + บริเวณที่ระบาย

9

Export

กดปุ่ม Export ที่แถบด้านบน หรือปุ่ม Export ในหน้า Preview:

  • ภาพเดี่ยว → บันทึกเป็น PNG
  • PDF หลายหน้า → เลือกหน้าที่ต้องการ แล้ว Export เป็น PDF ไฟล์เดียว
10

ทำงานกับ PDF

เมื่อเปิดไฟล์ PDF จะแสดงแผง filmstrip ด้านซ้ายพร้อม thumbnail ทุกหน้า:

การนำทาง ▲ ไป ▼
  • คลิกหน้า เพื่อเปิดหน้านั้นแปล
  • กดวงกลมมุมบนขวา ของ thumbnail เพื่อเลือก/ยกเลิกหน้าสำหรับ Export
  • ใช้ปุ่ม ▲▼ หรือพิมพ์เลขหน้าแล้วกด ไป
